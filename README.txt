== Description ==

Generate QR Codes for your Posts, Pages and Custom Post Types.

== Installation ==

= With Plugins Menu =
1. Open the 'Plugins' menu in WordPress
2. Search plugin by name 'SteeplyQR'
3. Install and Activate the plugin
4. Visit 'SteeplyQR' Page

= Manual Installation =
1. Upload `steeply-qr` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit 'SteeplyQR' Page

== Frequently Asked Questions ==

= How styling image from [sqr] shortcode? =

Use class `.sqr-img` in your styles file.

= How display QR Code in Page or Post? =

Use [sqr] shortcode.
